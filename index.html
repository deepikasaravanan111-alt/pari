Gradient Boosting Classifier

sklearn GradientBoostingClassifier trained on Add Health Wave IV (n = 6,504). Hyperparameters: n_estimators=300, max_depth=4, learning_rate=0.05, subsample=0.8. Evaluated via 5-fold stratified cross-validation. AUC = 0.885. Replicated across XGBoost, random forest, and regularized logistic regression. Model served live via FastAPI backend with joblib serialization.

Causal Mediation Analysis

Total COMT effect decomposed into direct and indirect paths using potential outcomes framework. Indirect effect runs through inhibitory control failure (H4MH22). Bootstrapped server-side (200 resamples per request) to produce real-time 95% confidence intervals on path coefficients. Indirect effect: 67% of total (95% CI: 54% to 79%, p less than 0.001).

Permutation Feature Importance

Computed server-side by permuting each feature in the user's specific input vector against population means from the Add Health training set. Produces a personalized importance profile rather than average training-set importance. 20 permutation repeats per feature. Results returned directly from the real gradient boosting model.
